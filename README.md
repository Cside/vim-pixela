# vim-pixela

Vim plugin grows your grasses with starting Vim.

![vim-pixela](https://pixe.la/v1/users/mattn/graphs/vim-pixela)

## Usage

```
let g:pixela_username = 'XXX'
let g:pixela_token = 'XXX'
```

## Requirements

curl command, openbrowser.vim
